> 1.8万家店、5万目标、每杯亏1.2元。库迪不是在做咖啡，是在用咖啡做一场对瑞幸的复仇。但组织不能建立在复仇之上。

## 一、复仇者的诞生

2022年，陆正耀被自己创立的瑞幸扫地出门两年后，做了两件事：

第一件，在厦门注册了「库迪咖啡」。
第二件，把瑞幸的核心班底——原CTO、原供应链负责人、原加盟管理负责人——全部挖了过来。

这不是创业。这是**复仇**。

库迪的英文名「Cotti」据传来自「Coffee + OTTI（意大利语"八"）」，但所有人都知道真实含义：**陆正耀要用库迪证明，瑞幸的成功是他个人的成功，不是郭谨一的成功。**

| 时间线 | 事件 |
|--------|------|
| 2020.04 | 瑞幸曝22亿财务造假，陆正耀出局 |
| 2022.10 | 库迪首店在福州开业 |
| 2023.02 | 库迪发起「全场9.9元」价格战 |
| 2023.05 | 7个月2500家店 |
| 2024.10 | 全球门店破10000家 |
| 2025.12 | 门店突破18000家 |
| 2026.01.31 | 9.9元活动正式结束 |
| 2026.02 | 库迪出现净关店 |

## 二、「寄生店」：陆正耀的组织创新

库迪的扩张速度打破了全球餐饮连锁的所有纪录——10个月开出6000家店。

秘密武器是一种独特的扩张模式：**「店中店」**——在别人的门店里嵌入一个6-12平米的咖啡档口。

合作方包括：
- **华莱士**：全国18000+家汉堡店
- **美宜佳**：全国30000+家便利店
- **苏宁易购**：全国2000+家电器卖场

逻辑很简单：不用租店面、不用装修、不用招人——合作方提供已有空间和人力，库迪提供设备和原料。开店成本从几十万降到几万。

这是陆正耀最擅长的东西：**用结构创新降低扩张门槛。**

但「寄生店」有一个致命缺陷：**品牌不存在。**

当你从华莱士的汉堡柜台旁接过一杯库迪咖啡，大脑里存储的不是「库迪」这个品牌，而是「华莱士旁边卖的便宜咖啡」。品牌寄生在别人的空间里，等于放弃了建立独立品牌认知的可能。

## 三、数学不会骗人

库迪的核心成本结构是一个公开的秘密：

| 成本项 | 每杯金额 |
|--------|:-------:|
| 原料 | 5.7元 |
| 包材 | 1.5元 |
| 人工 | 1.9元 |
| 水电 | 0.2元 |
| 房租 | 1.8元 |
| **合计** | **11.1元** |

售价9.9元。**每卖一杯亏1.2元。**

对比瑞幸：在3万家店的规模效应下，单杯成本已压至8.5元，9.9元促销仍能维持微利。

库迪为什么做不到？因为瑞幸花了5年建设供应链：自建烘焙基地（年产8万吨）、巴西直采协议（5年24万吨）、数字化中台。库迪只有两年。供应链的积累不能用钱买，只能用时间换。

还有一个数字更具说服力：

| 指标 | 瑞幸 | 库迪 |
|------|:----:|:----:|
| 单店日均杯量 | 350+ | ~100-120 |
| 盈亏平衡点 | ~200杯 | ~300杯 |

库迪不仅要亏每杯1.2元，而且门店日均杯量只有盈亏平衡点的1/3。这意味着联营商（加盟商）的成本结构中还要加上**持续的运营亏损**。

难怪据统计，库迪已有超过15000家门店关闭或转让。联营商维权事件频发。

## 四、一个组织的灵魂问题

所有伟大的公司都有一个共同的起点：创始人看到了一个**未被满足的需求**。

陆正耀看到了什么？**一个复仇的机会。**

这不是道德评价，这是组织分析。当创始人的驱动力量是「证明别人错了」而不是「解决一个真实问题」，这个组织的决策逻辑会产生系统性偏差：

1. **为增速牺牲盈利**：不是为了用户需求，是为了「打败瑞幸」
2. **用加盟商的钱做增长实验**：0加盟费+低保证金，加盟商承担亏损
3. **品牌建设是事后补丁**：便利店→咖啡店→热食档口，业态一变再变
4. **创始人风险转移给系统**：陆正耀个人被执行金额约30亿，公司资金链紧张

2026年2月，库迪宣布「全场9.9元」活动结束。不是不想打价格战了——是**打不起了**。同月出现净关店。

2026年6月，库迪宣布进军便利店模式：增加冰淇淋、烤肠、热食。从「咖啡店」变成「便利店里有咖啡」。

这不像战略升级。这像**慌不择路**。

## 五、库迪的三种结局

### 结局A：「止血存活」

关掉一半亏损门店，聚焦核心城市直营，从「规模第一」转「盈利第一」，收缩到5000-8000家健康门店。需要彻底否定当前的扩张逻辑。陆正耀做得到吗？

### 结局B：「便利店转型成功」

「咖啡+热食+便利店」的混合业态在中国市场找到了自己的位置，不是瑞幸的替代品，而是一个新品类。但这需要完全不同的组织能力——供应链从咖啡扩展到食品、门店运营从档口扩展到大店。库迪现在的组织骨架撑不住。

### 结局C：「资金链断裂」

最可能的结果。当新店增速放缓→加盟费收入下降→无力补贴联营商→更大规模的关店和维权→品牌崩塌。这是负反馈循环的经典剧本。陆正耀的历史——瑞幸、神州租车、神州优车——让「结局C」的权重天然偏高。

## 六、结语：组织不能建立在复仇之上

库迪的故事是一个组织学上的悖论：

**陆正耀用他当年做瑞幸的同一套方法论——极速扩张、价格炸弹、加盟杠杆——试图证明瑞幸的成功是他的方法论的成功。**

但如果这套方法论真的可持续，为什么他需要用库迪来证明，而不是瑞幸本身就证明了？

库迪的困局不是「咖啡行业太卷」，而是**复仇驱动型组织有天然的边界**——当「打垮瑞幸」比「服务好客户」更重要，组织做出的每一个决策都在偏离商业本质。

2026年大概率是库迪的命运分水岭。不是瑞幸打败了库迪——是**重力打败了库迪**。
